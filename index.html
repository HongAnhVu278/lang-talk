I was talking to Grace (SP1 2026), and she was learning Mandarin at the time.

She gave me some advice.

grace

When you learn a new word in a new language, you shouldn't translate it into your native language. You should learn it on its own terms.

me

omg that's so true 😱

Okay, now back to the programming story.

So, when I first got into programming in college, I learned Java.

And it's very object-oriented. Classes, inheritance, blahblah...

When I started learning Go, I tried to force that OOP mindset onto it.

And it was frustrating, because Go just isn't like that…?

And then I remembered what Grace said 🤯 (!!!!)

I'd been translating the whole time.

I should have been learning Go the Go way, instead of forcing an existing system of logic onto it.

It's interesting to me how I ended up using a skill from learning a natural language to learn a programming language.

Use the language!
